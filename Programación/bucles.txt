-BUCLES de RECORRIDO FIJO: Cuantas veces van a repetir el código.
--FOR.
---SINTAXIS DE FOR: 
----for (int i=0 ; i<3 ; i++){
	Todo lo que está aquí se repetirá en base a lo que esté declarado en los paréntesis.
----}

-int i=0 -> CONTADOR
-i < 3 -> CONDICIÓN
-i++ -> INCREMENTO O DECREMENTO.
Este código se ejecutará 3 veces y termina el bucle.
i, j, k, l
La i significa las veces que se repetirá el bucle.
-------------
RECORRIDO VARIABLE: cuando no sé cuántas veces se va a repetir el código. Se cumple mientras
se cumpla una condición.
WHILE: Mientras se cumpla la condición.
DO: Ejecuta el código hasta que se cumpla una condición.
UNTIL: Hasta que se cumpla la condición.
